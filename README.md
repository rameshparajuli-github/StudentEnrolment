# Student Enrolment System

## Description
A simple Student Enrolment System built using Java and 
Object-Oriented Programming principles.
This was developed as part of ITS206 - Software Construction and Design.

## Features
- Enrol students into courses
- Waitlist system when a course is full
- Display student, course and enrolment information

## Classes
- **Student** - stores student ID, name and email
- **Course** - stores course details and manages enrolment
- **Enrolment** - links a student to a course with a date
- **Main** - runs the program and demonstrates all features

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`
